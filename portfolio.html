Projects

Easy Animator with Java

Uses text file inputs that define initial and end values for shape, time, dimensions, position, and RGB color to generate animations. The program supports outputting animations as SVG files and within an interactive GUI with the following functionalities: Play, Pause, Loop, Restart, Discrete Time, Toggle Shape Fill, and a Speed Slider.

Shortest Path Maze Solver with C++

Solves text file mazes and displays the shortest path to the goal. It demonstrates the use of breadth-first search (BFS) and then Dijkstra's algorithm for each maze.

- The + is navigating the maze
- The * is the goal
- The X's are walls

Image Segmentation with MATLAB

Used GMM-based clustering to segment color images in MATLAB. This process included preprocessing the image, maximum likelihood estimation, 10-fold cross validation using BIC for model selection, fitting a new GMM and segmenting the image with the number of Gaussian components.

Mapping Coral Bleaching with Python and SQL

The interactive maps utilize an SQL database with coral bleaching data from across the globe. Data entries include location information, the source the reported it, coral species, coral family, percentage affected, water temperature, and metrics for bleaching severity, mortality and recovery.

Single-Cycle MIPS Processor with Verilog

Performs instruction sets of machine code that were translated from MIPS assembly language. Designed with behavioral Verilog code to establish the data path and generate the following components: Program Counter, Instruction Memory, Instruction Decoder, Register, ALU, Multiplexers, Data Memory, and a Debouncer.
